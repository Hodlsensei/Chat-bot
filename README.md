# Mrs. Lillian Visa Assistant

A WhatsApp-style chatbot to help clients get visa information before their consultation with Mrs. Lillian.

## Features
- 📱 WhatsApp-style interface
- 🎯 Smart visa knowledge base
- 💬 Interactive chat experience
- ⚡ Quick reply buttons
- 📞 Direct WhatsApp integration

## Setup
1. Clone this repository
2. Install dependencies: `npm install`
3. Update contact info in `.env.local`
4. Run locally: `npm run dev`
5. Deploy to Vercel: `vercel`
